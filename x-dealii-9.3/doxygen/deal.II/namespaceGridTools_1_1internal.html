A function that fills DistributedComputePointLocationsInternal. If the input argument perform_handshake is set to false only the fields needed by GridTools::internal::distributed_compute_point_locations() are filled. If the input argument is set to true additional data structures are set up to be able to setup the communication pattern within Utilities::MPI::RemotePointEvaluation::reinit().
